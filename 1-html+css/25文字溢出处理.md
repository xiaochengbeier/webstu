# 文字溢出处理
【小城贝尔】

单行溢出三件套，静止换行溢出到。
多行处理仅截断，容器文字高计算。

## 单行溢出三件套，静止换行溢出到。
    单行文本溢出处理方法一般是打点展示
    div {
        white-space: nowrap;
        /*表示多余不换行*/
        overflow: hidden;
        /*溢出部分隐藏*/
        text-overflow: ellipsis;
        /*打点展示*/
   }
## 多行处理仅截断，容器文字高计算。
    多行在 pc端仅做截断处理 需要计算好容器高度和行高之间的关系避免截断半截的情况
