类型校验跟冒号，数据字符后续到。

## 类型校验跟冒号，数据字符后续到。
   常用的类型校验使用 :类型即可
## 遇到公有别名号，类型字符变量到。
   有时候一些类型是复杂数据类型的并且是公用的这个时候我们
   就可以使用 
   type 类型名 定义类型别名提高复用性
```ts
    type user = {
        name:string,
        age:number
    }
    let user1:user;
    user1={name:"用户",age:20};
```
